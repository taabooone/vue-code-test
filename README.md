# vue_code_test

> code test project for vue.js

## Build Setup

``` bash
# install dependencies
npm install
cd server
npm install

# start server
npm run dev

# serve with hot reload at localhost:8080
cd ..
npm run dev

# create a teamlead on the server

POST localhost:3001/users 

{
  email: 'test@example.com',
  password: 'password',
  role: 'team-lead'
}
```
